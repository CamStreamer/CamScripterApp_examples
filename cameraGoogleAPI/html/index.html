Camera Remote API

Changes are saved automatically, you don't have to save them.


HOW TO SET UP Camera Remote API script Step 1: Create the Google Sheet and click File > Publish to the web > Publish
Step 2: Share also publicly the sheet (via the Share button in the top right corner) so CamScripter can read from it and also allow editing your sheet with the people you want to change the values in the video remotely.
Step 3: Copy unique Google Sheet ID from sheet URL: e.g. https://docs.google.com/spreadsheets/d/abcde12345BpVoRMQv3J_6yFpSWuSwtRf5QnyqT_qyOI/edit#gid=0 (the bold part) and Sheet name
Step 4: Generate the Google Sheet API key on https://console.cloud.google.com/apis/credentials and copy and paste the API key. Please note, there are limits of requests per Google Account. Check Google Sheet API support for the current limits.
Step 5: Eneble Google Sheet API on https://console.cloud.google.com/apis/api/sheets.googleapis.com/overview
Step 6: Enter API requests you want to trigger with localhost IP 127.0.0.1

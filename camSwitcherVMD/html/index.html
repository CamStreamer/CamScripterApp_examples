CamSwitcherVMD

This script allows automatic switching between 2 cameras based on VMD 4 installed on both cameras. CamSwitcher App needs to be installed on Camera 1 and both cameras have to be added and present in the CamSwitcher App so this script can switch the view when motion is detected on any camera. If no motion is detected by VMD 4 ACAP, the script will switch between cameras in the set period.

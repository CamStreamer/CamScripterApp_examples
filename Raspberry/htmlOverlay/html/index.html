CamScripter

HTML Overlay

This package converts HTML webpage to an image and sends the image to the CamOverlay application.

The package requires Chromium and node.js version 14 or newer.
Chromium installation: sudo apt install chromium-browser.
Node.js upgrade:

  • sudo npm install -g n
  • sudo n latest
  • sudo systemctl restart camscripter.service

Changes are saved automatically, you don't have to save them.
